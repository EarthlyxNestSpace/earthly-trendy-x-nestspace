## 📦 How to Run / Deploy
**GitHub Pages**
1. Repo → Settings → Pages → Source: `Deploy from a branch`
2. Branch: `main` • Folder: `/ (root)` → Save
3. Wait ~1–2 minutes and open the live link above.
